# SpringBoot  X-bi

> 作者：X
传统 BI 平台需要按照以下步骤[]：
> 
> 1.手动上传数据
> 2 .手动选择分析所需的数据行和列（由数据分析师完成）
3 需要手动选择所需的图表类型（由数据分析师完成）
4 生成图表并保存配置
> 本项目：允许用户（数据分析者）仅需导入最最最原始的数据集并输入分析目标（例如网站增长趋势），
> 即可利用 AI 自动生成符合要求的图表和结论，从而显著提升分析效率。
> 优点：即使没有数据分析经验的同学也能通过简单的输入目标快速完成数据分析，利用 AI 接口来实现这一功能
